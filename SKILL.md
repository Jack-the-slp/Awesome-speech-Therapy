## 专业术语体系

### 构音（Articulation）
语音通过口腔、鼻腔等声道器官的运动而形成的过程。构音障碍指发音不准确、不清晰。

### 声母（Consonant）
发音时气流在声道中受到阻碍的音。普通话21个声母。

### 韵母（Vowel）
气流通过声道不受阻碍的音，或气流在口腔中受到轻微阻碍的音。

### 构音障碍评估六维度
1. **呼吸系统** — 呼吸方式、呼吸支持能力
2. **发声系统** — 嗓音质量、音调、响度
3. **共鸣系统** — 鼻腔共鸣、口腔共鸣
4. **口部运动能力** — 下颌/唇/舌的运动控制
5. **构音能力** — 声母和韵母的发音准确性
6. **语言流畅度** — 正常/一字一顿/口吃

## 心智模型

### 核心框架：TC-分步式构音教学法

**三步错误分析法**（所有评估和训练的核心逻辑）:

**Step 1 — 正确的音如何发出**
- 目标音的声道位置（哪里）
- 气流如何被阻碍/调节（怎么动）
- 声带是否振动（清/浊）

**Step 2 — 错误的音如何错的**
- 扭曲（D）: 音被发出但部位或方式不准确，轻/中/重表示严重程度
- 替代: 用另一个音代替目标音（如 g → d）
- 省略: 音被完全省略（如"星星"说成"西西"）
- 添加: 添加了不该有的音

**Step 3 — 选择怎么样的方法**
- 根据错误类型选择训练策略
- 遵循"先易后难"原则

**完整案例示范**:
> **案例：g音错误（g → d 替代）**
> - Step 1 正确的音如何发出：舌根向后上抬，抵住软硬腭交界，形成阻塞后突然释放，g是清辅音
> - Step 2 错误的音如何错的：舌根位置偏前，舌尖参与了接触，变成了d的舌尖-齿龈位
> - Step 3 选择怎么样的方法：视觉提示舌根位置（压舌板辅助）+ 最小对立项d-g对比 + 舌根向后收缩训练

### 评估优先级决策树

```
主诉是什么？（言语不清晰 / 嗓音沙哑 / 流涎 / 喂养困难）

├─ 言语不清晰 → 进入构音能力评估
│   ├─ 先评声母（21个）→ 找出错误模式
│   └─ 再评韵母（28个）→ 确认元音是否受影响
│
├─ 嗓音沙哑 → 进入发声系统评估
│   └─ 嗓音质量 / 音调 / 响度
│
├─ 流涎 → 进入口部运动评估（下颌稳定性）
│
└─ 喂养困难 → 优先评估呼吸-吞咽协调性
```

## 声母分类体系（按解剖位置）

### 双唇音（b / p / m）
- b: 双唇紧闭，清辅音，气流完全阻塞后突然释放
- p: 同b，送气版本
- m: 双唇紧闭，浊辅音，鼻音

### 舌尖音（d / t / n / l）
- d/t: 舌尖抵上齿龈，清辅音（t送气）
- n: 舌尖抵上齿龈，浊辅音，鼻音
- l: 舌尖抵上齿龈，浊辅音，边音（气流从舌侧流出）

### 舌根音（g / k / h）
- g/k: 舌根抵软硬腭交界，清辅音（k送气）
- h: 舌根接近软腭，清辅音

### 舌面音（j / q / x）
- j: 舌面前部贴硬腭，清辅音
- q: 同j，送气版本
- x: 舌面前部接近硬腭，清辅音

### 舌尖前音（z / c / s）
- z/c/s: 舌尖抵上齿背，清辅音（c送气）

### 舌尖后音（zh / ch / sh / r）
- zh/ch/sh: 舌尖翘起抵硬腭前部（ch送气，sh清，r浊）
- r: 舌尖后音，浊辅音

### 唇齿音（f）
- f: 上齿抵下唇内侧，清辅音

- ang / eng / ing / ong（后鼻音，舌根抵软腭）

## 呼吸系统评估

### 呼吸模式分类
| 类型 | 特征 | 临床意义 |
|------|------|---------|
| 胸式呼吸 | 胸廓扩张为主 | 正常成人模式；幼儿可能提示呼吸浅 |
| 腹式呼吸 | 腹部扩张为主 | 深层呼吸，支持好 |
| 胸腹联动 | 胸腹同步扩张 | 最理想的言语呼吸模式 |

### 评估指标
- **最长声时（MPT）**: 深吸气后最长持续发声时间
  - 成人正常值：男性20-30秒，女性15-20秒
  - 儿童随年龄递减
- **言语时呼吸控制**: 能否在啭音练习中维持稳定呼气流

### 呼吸支持不足的训练
- 腹式呼吸训练（仰卧位吹泡泡）
- 最长声时延长训练
- 逐字句呼吸控制训练

## 发声系统评估

### 嗓音质量
| 类型 | 描述 | 常见人群 |
|------|------|---------|
| 正常 | 无异常 | — |
| 嘶哑声 | 嗓音粗糙 | 声带病变、过度用声 |
| 气息声 | 呼气漏气感明显 | 声门闭合不全 |

### 音调评估
- 音调正常 / 音调过高（常见于痉挛型脑瘫）/ 音调过低

### 响度评估
- 音量正常 / 音量过高 / 音量过低（弛缓型常见）

## 共鸣系统评估

### 鼻腔共鸣
- **正常**: 鼻音和非鼻音界限清晰
- **亢进**: 过多气流从鼻腔流出（常见于腭裂术后）
- **低下**: 鼻音缺失（感冒时常见）

### 口腔共鸣聚焦
- **前位聚焦**: 音调偏尖细，常见于硬腭高窄
- **后位聚焦**: 音调沉闷，常见于舌根向后用力
- **喉位聚焦**: 音调压低，嗓音紧压感
- **正常**: 口腔共鸣均衡

## 口部运动能力评估（关键！）

### 下颌评估

#### 咬肌肌力
- 正常 / 较弱

#### 模仿动作
- 上 / 下 / 左 / 右 四个方向
- 上下交替运动（如a-i-a-i）
- 左右交替运动

#### 言语状态下的分级控制
**两级控制**（相邻元音）:
- a-i / a-e / i-e 交替，如 a-i-a-i-a-i

**三级控制**（三个元音）:
- a-i-e 交替，如 a-i-e,a-i-e,a-i-e
- a-e-i 交替，如 a-e-i,a-e-i,a-e-i

**下颌半开位控制**: 下颌处于半开状态时的稳定性和控制能力

#### 下颌功能训练原则
1. 先提高肌力（咬棒抵抗训练）
2. 再练模仿动作（视觉指令）
3. 然后交替运动（韵律训练）
4. 最后言语状态分级控制（元音组合练习）

### 唇评估

#### 静态评估
- 肌力：正常 / 较弱
- 肌张力：正常 / 偏高 / 偏低

#### 模仿动作
- 闭合（m）/ 圆唇（u）/ 唇齿（f）/ 展唇（i）/ 翘唇（r）
- 交替运动：u-i-u-i（圆展交替）、a-o-a-o（唇角内收）

#### 言语状态下评估要点
1. /m/ 闭唇：是否准确、自然、无力、代偿
2. /u/ 圆唇：是否准确、紧张、不能完成
3. /f/ 唇齿：是否准确、双唇代偿、唇齿模式未习得
4. /i/ 展唇：是否准确、不会
5. /r/ 翘唇：是否准确、代偿、不会
6. 交替动作连贯性

### 舌评估

#### 外观检查
- 舌分段：前 / 中 / 后 / 两侧 / 舌叶
- 外观：正常 / 偏移（L/R）
- 舌系带：正常 / 偏短 / 过短（影响舌尖上抬）

#### 运动能力
- 方向：前 / 后 / 左 / 右
- 交替：左右交替、上下交替
- 上抬：前部上抬、后部上抬、舌两侧上抬
- 舌面上抬：马蹄形上抬（感知）

#### 分化程度
- **粗大运动分化**: 舌能做大幅度的方向运动
- **精细运动分化**: 舌能做精细的局部运动

#### 稳定性
- 稳定 / 不稳定（舞动、扭曲）

### 腭评估

#### 软腭
- 高度：正常 / 偏低（悬雍垂分叉提示异常）
- 运动：正常 / 节律性波动或痉挛（言语时悬雍垂观察）

#### 腭咽闭合
- 正常 / 腭咽闭合不全（VP closure）

#### 功能测试
- 吹气：是否鼻漏气
- 鼓腮：是否鼻漏气

## 构音错误记录规范

### 扭曲（D）
- 使用字母"D"并标注程度：1（轻度）、2（中度）、3（重度）
- 例：`b D1`（b轻度扭曲）

### 替代
- 记录错误走向
- 例：`g → d`（g被d替代）

### 可诱发（+）
- 表示在辅助下可以发出该音
- 例：`g +`（g可诱发）

### 完全省略
- 目标音完全未出现
- 例：`省略 s`（s音被省略）

## 决策启发式（临床决策规则）

### H1: 构音评估优先顺序
1. 先收集基本信息（出生史/发育史/喂养史/诊断史）
2. 再评呼吸-发声-共鸣（基础生理）
3. 再评口部运动（运动基础）
4. 最后评构音能力（最终表现）

### H2: 错误类型 → 训练策略映射

| 错误类型 | 训练策略 |
|---------|---------|
| 扭曲（D1轻度） | 听觉感知训练 + 视觉提示 |
| 扭曲（D2中度） | 口部运动基础训练 + 目标音诱发 |
| 扭曲（D3重度） | 从基础口部运动重新建立 |
| 替代（g→d） | g音的口部定位训练（舌根抬升） |
| 省略（s省略） | 气流持续训练 + s音的舌尖定位 |
| 可诱发（+） | 最小对立项训练 + 强化练习 |

### H3: 下颌优先原则
- 很多唇音/舌音的问题根源在下颌控制不足
- 遇到唇/舌问题，先查下颌稳定性
- 下颌两级控制稳定后，再进入三级控制

### H4: 感知先于发音
- 先确认患者能**听到**两个音的差别（听觉辨别）
- 再训练**说出**正确音（构音动作）
- 顺序颠倒会导致训练效率低下

### H5: 家庭训练权重
- 每周1-2次治疗远远不够
- 必须给家长具体的每日家庭训练任务
- 目标：让家庭成为"半个治疗师"

## 训练实施规范

### 分步式训练流程
**每个目标音的训练都遵循以下步骤**:

**阶段1 — 感知建立**
- 目标：让孩子听到正确音和错误音的差别
- 方法：最小对立项听觉辨别训练

**阶段2 — 口部定位**
- 目标：在静息状态下建立目标音的口部姿势
- 方法：视觉提示 + 触觉提示 + 镜子反馈

**阶段3 — 在单音/音节中诱发**
- 目标：将口部姿势与声音结合
- 方法：目标音+核心韵母组合（如b→ba/bi/bo）

**阶段4 — 在词语中稳定**
- 目标：在实际词汇中维持目标音
- 方法：图片命名 → 短语 → 句子

**阶段5 — 对练转移**
- 目标：在对话中自然使用
- 方法：情景对话 + 家庭泛化

## 常见声母训练要点

### b/p/m（双唇音）
- 关键：双唇闭合的力度和时间控制
- 常见错误：双唇无力→用m代替b/p；双唇不闭合→用u/o代替

### d/t/n/l（舌尖音）
- 关键：舌尖抬升的力度和位置
- 常见错误：舌尖后缩→用g/k代替；舌尖力量不足→省略

### g/k/h（舌根音）
- 关键：舌根向后上抬
- 诱发方法：压舌板轻压舌中部，让患者发声

### z/c/s（舌尖前音）
- 关键：舌尖抵上齿背，气流持续
- 常见错误：舌尖位置偏后→与zh/ch/sh混淆

### zh/ch/sh/r（舌尖后音）
- 关键：舌尖翘起抵硬腭前部
- 诱发方法：舌边吸引（用勺子边缘引导舌尖上抬）

### f（唇齿音）
- 关键：上齿抵下唇内侧，气流摩擦
- 常见错误：下颌前伸代偿→齿唇接触不准确

## 常见韵母训练要点

### 单元音 a/i/u/ü
- a：口完全张开，舌居中
- i：唇展扁，舌尖抵下齿
- u：唇收圆，舌根高
- ü：唇收圆+展扁组合

### 复韵母
- 核心难点：从一个元音滑向另一个元音的过程
- 训练重点：气流不能断，滑音过程要连贯

### 鼻韵母
- an/en/in/un/ün：前鼻音，舌尖抵上齿龈
- ang/eng/ing/ong：后鼻音，舌根抵软腭
- 常见错误：前后鼻混淆（an→ang）

```python
def create_table(doc, data, col_widths=None):
    """创建简洁表格"""
    table = doc.add_table(rows=len(data), cols=len(data[0]))
    table.alignment = WD_TABLE_ALIGNMENT.CENTER
    table.style = 'Table Grid'
    for i, row_data in enumerate(data):
        for j, text in enumerate(row_data):
            cell = table.rows[i].cells[j]
            cell.text = str(text)
            cell.vertical_alignment = WD_ALIGN_VERTICAL.CENTER
            for para in cell.paragraphs:
                para.paragraph_format.space_before = Pt(6)
                para.paragraph_format.space_after = Pt(6)
                for run in para.runs:
                    run.font.name = '微软雅黑'
                    run.font.size = Pt(10)
                    if i == 0:
                        run.bold = True
            set_cell_border(cell)
    if col_widths:
        for row in table.rows:
            for idx, width in enumerate(col_widths):
                row.cells[idx].width = Cm(width)
    return table

def add_heading1(doc, text):
    """一级标题 15pt 加粗"""
    p = doc.add_paragraph()
    p.paragraph_format.space_before = Pt(20)
    p.paragraph_format.space_after = Pt(8)
    run = p.add_run(text)
    run.font.name = '微软雅黑'
    run.font.size = Pt(15)
    run.bold = True
    return p

def add_heading2(doc, text):
    """二级标题 12pt 加粗"""
    p = doc.add_paragraph()
    p.paragraph_format.space_before = Pt(14)
    p.paragraph_format.space_after = Pt(6)
    run = p.add_run(text)
    run.font.name = '微软雅黑'
    run.font.size = Pt(12)
    run.bold = True
    return p

def add_para(doc, text, bold=False, indent=True):
    """普通段落 11pt"""
    p = doc.add_paragraph()
    p.paragraph_format.space_before = Pt(4)
    p.paragraph_format.space_after = Pt(4)
    if indent:
        p.paragraph_format.left_indent = Inches(0.3)
    run = p.add_run(text)
    run.font.name = '微软雅黑'
    run.font.size = Pt(11)
    run.bold = bold
    return p

def add_bullet(doc, text, indent=0.5):
    """项目符号"""
    p = doc.add_paragraph(style='List Bullet')
    p.paragraph_format.left_indent = Inches(indent)
    p.paragraph_format.space_before = Pt(3)
    p.paragraph_format.space_after = Pt(3)
    run = p.add_run(text)
    run.font.name = '微软雅黑'
    run.font.size = Pt(11)
    return p
```

**标准文档结构**:

1. **封面**: 标题 + 副标题 + 报告信息表
2. **一、问题定性**: 错误类型 + 错误走向
3. **二、TC-三步错误分析法**: Step1 + Step2 + Step3
4. **三、训练方案**: 分阶段训练计划表格
5. **四、家庭训练**: 任务卡表格
6. **五、注意事项**: 要点列表
7. **六、理论支持**: 参考文献列表
8. **签名**: 分析者和日期

**输出路径**: 桌面 `os.path.join(os.path.expanduser("~"), "Desktop", "文档名.docx")`

**页面设置**: 上2.5cm、下2.5cm、左3cm、右3cm
