## 核心方法论

### TC-三步错误分析法

```
Step 1 ─── 正确的音如何发出
          声道位置 / 气流调节 / 清浊辅音

Step 2 ─── 错误的音如何错的
          扭曲(D) / 替代(/→) / 省略 / 添加

Step 3 ─── 选择怎么样的训练方法
          先易后难 / 感知先行 / 家庭协同
```

### 六维度评估体系

| # | 维度 | 评估内容 |
|---|---|---|
| 1 | 呼吸系统 | 呼吸方式、最长声时、言语时呼吸控制 |
| 2 | 发声系统 | 嗓音质量、音调、响度 |
| 3 | 共鸣系统 | 鼻腔共鸣、口腔共鸣聚焦 |
| 4 | 口部运动 | 下颌/唇/舌/腭的运动控制能力 |
| 5 | 构音能力 | 声母（21个）+ 韵母（28个）发音准确性 |
| 6 | 语言流畅度 | 正常 / 一字一顿 / 口吃 |

### 分步式训练流程

```
感知建立 ──→ 口部定位 ──→ 诱发 ──→ 稳定 ──→ 泛化
   ↑                                            ↑
   └────────────── 循环巩固 ──────────────────────┘
```
